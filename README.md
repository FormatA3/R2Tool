# Resistance 2 - Tool

Cet outil permet :
- Save Editor (basic avec calcul CRC32).
- DÃ©compression/compression des fichiers PSARC (listing des fichiers dans "fileslist.txt")
- Extraction des Assets via assetlookup.dat et highmips.dat (DDS et IRB).
- Editeur de texture (Preview, extraction et remplacement - DDS)
- Compilateur de fichier LUA pour Resistance 2.

Utilisation de ICSharpCode.SharpZipLib.dll et de LuaCompilerR2.dll (Modification des fichiers source Lua 5.1.2 pour que ce soit compatible pour PS3 et le bon format pour Resistance 2).
